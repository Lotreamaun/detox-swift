# Detox — План разработки MVP

## Описание проекта

**Назначение:** Приложение заметок для личного использования (не для App Store)
**Платформа:** iOS (SwiftUI)
**Автор:** Владимир Пудовкин

---

## Требования

### Функционал
- Создание/сохранение/редактирование/удаление заметок
- Форматирование: жирный, курсив, подчеркнутый, highlight, спойлер
- Списки: нумерованные, маркированные, todo-чеклист
- Markdown: парсинг, экспорт в .md

### Данные
- Сохранение после закрытия (persistence)
- Общая БД с веб-сервисом на Python + JS/TS

---

## Этапы разработки

### Этап 1: Базовая функциональность

1.1. Показать список заметок из NotesData в NoteList
1.2. Добавить навигацию: список → редактор
1.3. Кнопка "+" для создания новой заметки
1.4. Сохранение в UserDefaults

**Результат:** Приложение создает, показывает, редактирует и сохраняет заметки.

---

### Этап 2: Удаление и базовый UI

2.1. Удаление заметки (свайп влево)
2.2. Кнопка "Назад" с автосохранением
2.3. Минимальный UI: заголовок заметки

---

### Этап 3: Форматирование текста

3.1. Кнопки B/I/U в NoteEditor
3.2. Поддержка NSAttributedString (уже есть TextView)
3.3. Highlight разными цветами
3.4. Спойлер-текст

---

### Этап 4: Списки и todo

4.1. Нумерованные списки
4.2. Маркированные списки
4.3. Чекбоксы (todo-листы)

---

### Этап 5: Markdown (опционально)

5.1. Парсинг Markdown → NSAttributedString
5.2. Редактор в Markdown-режиме
5.3. Экспорт в файл .md

---

## Принципы разработки

- **KISS** — делаем максимально просто
- **YAGNI** — не добавляем лишний функционал
- **Итерации** — маленькие шаги, проверка после каждого
